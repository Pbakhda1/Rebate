Manufacturer Bundle Builder

If a manufacturer offers an instant discount (ex: $500 off an oven), Rebate can work with the manufacturer to provide a list of discounted items in the same category (appliances, etc.). You choose more items → bigger bundle. Rebate charges a bundle fee (example: $50).

Concept: Manufacturer provides the discount instantly. Rebate organizes the bundle and charges a set bundle fee.
